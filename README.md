## 📁 Structure
Each challenge has its own folder:
- 📌 **FutureVera/**
  - `README.md` — Detailed writeup for this challenge
  - `commands.md` — All commands used during enumeration and exploitation
  - `images/` — Screenshots and evidence (e.g., scans, flags, steghide output)

## 📌 How to Use
1. Navigate to a challenge folder.
2. Read the `README.md` for a step-by-step walkthrough.
3. Refer to the `images/` folder for screenshots supporting the steps.

## 💡 Notes
- Keep screenshots named.
- Each writeup should document **trial and error**, tools used, and lessons learned.
- Optional: Add a small summary at the end of each writeup highlighting key takeaways.
